# 写在前面的话

这是一篇夹杂私货的书评，其实更像是拿《恋恋笔记本》和卡梅隆著名的电影《泰坦尼克号》做对比，毕竟他们都讲述了一种不可能的，跨越阶级的爱情。

**注意⚠️**：本文无关于浪漫爱情的描述，更多的是意识形态批判。

# 《恋恋笔记本》，作为意识形态的终极幻想

从齐泽克的视角来看，之所以称其为“终极幻想”，是因为它完美地执行了“**拜物教式的否认**”，并且掩盖了生命中两个最可怕的创伤性内核：阶级对抗与肉体的实在界。阶级对抗很好理解，在故事中，诺亚是底层无产阶级，是工人，而艾莉所在的家庭是资产阶级，是大资本家。这种阶级鸿沟是不可跨越的，暴力的，不可调和的。然而，电影只用了一个充满激情的夏日就冲淡了这一不可调和的差异。对于肉体的实在界，书中极大程度上美化了阿尔茨海默症，将这个实在界的入侵美化成了一个每天读书的仪式，试图用符号界来驯化实在界。并且电影结尾男主和女主在“爱情”的作用下共同奔赴死亡，这在现实来看是不可能的，但作为意识形态，他却是必须的。因此，说他是意识形态的终极幻想，因为他封闭了作为不可能的跨越阶级之爱的社会裂缝，封闭了作为疾病的身体裂缝，封闭了死亡的裂缝。

# 《泰坦尼克号》，意识形态的陷阱
想象一下，如果船顺利抵达纽约会发生什么？肉丝和杰克会面临真实的日常生活。贫穷、阶级差异、生活习惯的摩擦很快就会耗尽激情。几个月后，肉丝可能会发现杰克只是个无所事事、身无分文的流浪汉，而杰克也会厌倦肉丝的贵族脾气。他们的爱情会在庸俗的现实中慢慢腐烂。在这样的情况下，“**冰山**”就不再是灾难，而是作为救世主的角色。冰山的撞击恰逢其时——就在他们发生性关系并承诺私奔之后。灾难的发生，**将他们的爱情冻结在了最完美的“想象界”时刻，在理想层面得到永恒的救赎**。我们可以看到意识形态在这里如何有效运作，有两个表层，事故的魔力还有爱情的力量，但是他们全部都是陷阱。是降低我们注意力的东西，可以说是给我们打开了一个入口去准备迎接背后真正保守的信息——富人企图通过无情占有穷人的生命力来获得新生。有一个很好的细节证明了这一点，当肉丝发现杰克已经死了的时候，她说着“我不会放弃的，我不会放弃的·······”，然后她将杰克推开。杰克在这里担任的角色，我们讽刺地称他为消失的中介。从肉丝的角度出发，杰克就已经不是他自己了，他已经在某种程度失去了同一性，成了辩证法的扬弃对象，即一种概念性的“灰色”的东西，从而顺利的被缝进肉丝的“身体”里，完成肉丝作为资产阶级的新生。
# 结语
作为处在青春期的男性强迫症主体，我难免对爱情有不切实际的幻想，因此将对爱情的美好幻想投射到电影、小说中。但是这些毕竟不是长久之计，真正的解决方案是付诸实践，投身到爱情的实践中，即找个人谈恋爱。
